# あめときどき、ツール ☔

このリポジトリは、[jq3xgz.github.io](https://jq3xgz.github.io/) で公開しているウェブサイトのファイルです。
主に、GitHub Pages の動作テストを目的としています。

## サイト概要

ちょっとしたインタラクティブな機能を持つ、シングルページのウェブサイトです。

*   **あなたのこと**: あなたのIPアドレスやブラウザ情報（ユーザーエージェント）を表示します。
*   **いろの採集**: カラーピッカーで好きな色を選び、そのカラーコードを確認できます。

## 使用技術

このサイトは、以下のシンプルな技術だけで作られています。

*   HTML
*   CSS
*   JavaScript (外部ライブラリは使用していません)
